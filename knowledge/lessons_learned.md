# Lessons Learned

Canonical register of evidence-backed Atlas commercial lessons. Learning Intelligence owns this file. A lesson is an actionable interpretation of a recorded outcome, not an activity note.

## Evidence and maturity

- Evidence State: VERIFIED / ESTIMATED / HYPOTHESIS / UNKNOWN
- Maturity: ONE-CASE / EMERGING / VALIDATED / RETIRED
- Only VALIDATED lessons may be described as durable operating knowledge.
- Revisions preserve the stable Lesson ID and append dated evidence; history is never silently rewritten.
