# Checklist — Atlas CEO

## Non-research gate

- [ ] No web search was performed.
- [ ] No external source was opened.
- [ ] No research was conducted.
- [ ] No fact, probability, revenue or timing value was invented.
- [ ] Every factual statement traces to accepted internal evidence.
- [ ] Missing evidence was delegated, not researched by CEO.

## Input gate

- [ ] Market Opportunity report is accepted.
- [ ] Product Intelligence report is accepted.
- [ ] Manufacturer Hunter report is accepted.
- [ ] Manufacturer Due Diligence report is accepted.
- [ ] Buyer Intelligence report is accepted.
- [ ] Pricing Intelligence report is accepted.
- [ ] Competition Intelligence report is accepted.
- [ ] Outreach Intelligence report is accepted.
- [ ] Relevant databases are reconciled.
- [ ] Current workspace and prior decisions are read.
- [ ] Missing/not-applicable reports are explained.

## Evidence coherence

- [ ] Product scope is consistent across reports.
- [ ] Manufacturer identity/status is consistent.
- [ ] Buyer-demand thesis is consistent.
- [ ] Pricing model matches product/buyer/manufacturer route.
- [ ] Competition strategy matches the pricing and outreach position.
- [ ] Outreach claims are supported.
- [ ] Contradictions are resolved or block GO.
- [ ] Evidence age is acceptable or delegated for refresh.
- [ ] DRAFT reports were not used as binding evidence.

## Decision questions

- [ ] Should Atlas pursue is answered YES or NO.
- [ ] Why is explained.
- [ ] Expected business value is sourced or UNKNOWN.
- [ ] Probability of success is sourced or UNKNOWN.
- [ ] Time to first meeting is sourced or UNKNOWN.
- [ ] Time to first revenue is sourced or UNKNOWN.
- [ ] Commercial model is selected.
- [ ] Next commercial milestone is explicit.
- [ ] Authorized and prohibited actions are explicit.

## Commercial model

- [ ] Representation was considered.
- [ ] Commission was considered.
- [ ] Back-to-back was considered.
- [ ] Hybrid was considered.
- [ ] Other model is defined if selected.
- [ ] Stock/exclusivity is not approved without specific evidence and authority.
- [ ] Primary model maximizes expected long-term commercial value.

## Kill rule

- [ ] Weak evidence is considered.
- [ ] Poor margin is considered.
- [ ] Strong competition is considered.
- [ ] Buyer demand is considered.
- [ ] Capital requirement is considered.
- [ ] Technical risk is considered.
- [ ] Commercial fit is considered.
- [ ] Sunk effort did not influence the decision.
- [ ] Killed opportunity includes reason and next opportunity.
- [ ] WATCH includes evidence, owner, deadline and promotion/kill trigger.

## Resource allocation

- [ ] Maximum manufacturers to contact is set.
- [ ] Maximum buyers to approach is set.
- [ ] Priority order is set.
- [ ] Weekly commercial targets are set.
- [ ] Sprint objective is set.
- [ ] Research-task cap is set.
- [ ] Capital cap is set or UNKNOWN with owner.
- [ ] Review date is set.
- [ ] Kill trigger is set.
- [ ] One primary opportunity has execution focus.

## CEO priorities

- [ ] TOP 3 PRIORITIES are produced.
- [ ] TOP 3 RISKS are produced.
- [ ] TOP 3 OPPORTUNITIES are produced.
- [ ] NEXT COMMERCIAL ACTIONS are produced.
- [ ] Every action has owner, due date and completion evidence.
- [ ] Priorities increase expected commercial success.

## Confidence and status

- [ ] Five confidence components are scored.
- [ ] Weights total 100%.
- [ ] Confidence arithmetic is verified.
- [ ] GO Confidence Score ≥ 80.
- [ ] GO has no fatal blocker.
- [ ] GO has a viable model and executable next action.
- [ ] WATCH means NO for now.
- [ ] NO-GO means NO/kill.
- [ ] Status does not overrule evidence.

## Outputs

- [ ] Decision uses the next `decisions/DECISION_XXX.md`.
- [ ] Decision follows the controlled template.
- [ ] Decision finishes with exact FINAL DECISION block.
- [ ] Dashboard is updated.
- [ ] Active project is updated.
- [ ] Current sprint is updated.
- [ ] Next actions are updated when needed.
- [ ] Decision registry is updated when required.
- [ ] All operating surfaces agree.
- [ ] Internal links resolve.
- [ ] No unrelated content was overwritten.

## FINAL DECISION block

- [ ] GO / WATCH / NO-GO
- [ ] Confidence Score
- [ ] Commercial Impact
- [ ] Expected Revenue Impact
- [ ] Expected Time To First Revenue
- [ ] Owner
- [ ] Next Action
- [ ] Blocking Issues
